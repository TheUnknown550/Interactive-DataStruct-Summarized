## Topic: Stack

### Overview
Stacks are LIFO: push, pop at the top.

### Operations
- push: add to top
- pop: remove from top
- peek: read top without removing

### Use Cases
- function call stack
- undo/redo

### Complexity
- Push: O(1)
- Pop: O(1)
- Space: O(n)
